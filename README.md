## Getting Started 🏁

### Prerequisites 📦

#### Arch Linux 🐧
```bash
sudo pacman -S meson base-devel
```

#### Debian/Ubuntu 🐧
```bash
sudo apt install meson ninja-build
```

### Clone the Repository ⬇️
```bash
git clone https://github.com/silitonix/Matter.git
cd Matter
```

### Build the Project 🏗️
```bash
meson setup build # arch-meson for arch linux
meson compile -C build
```
